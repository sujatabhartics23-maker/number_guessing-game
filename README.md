# Number Guessing Game in C

This is a simple number guessing game built using C programming.  
The program generates a random number, and the user has to guess it in minimum attempts.

## Features
- Random number generation
- User input handling
- Feedback for each guess (too high / too low)
- Total attempts display

## How to Run
1. Compile the file:
   gcc guess.c -o guess

2. Run the program:
   ./guess

## Concepts Used
- Random number generation (rand, srand)
- Loops (while)
- Conditional statements (if-else)
- User input (scanf)

## Output Example

Guess the number between 1 and 100: Enter your guess: 45 Too low! Try again.
